## 客户角色工作报告

### 1. 行业背景与业务场景
- 选取线上消费信贷平台作为场景，平台通过撮合投资人与借款人完成信用借款交易。
- 核心痛点：逾期风控失效将直接影响投资人回款体验与平台口碑，需要在放款前给出违约风险评估。

### 2. 需求与指标
- **功能需求**：
  - 依据借款人与历史行为数据，预测借款人在整个还款周期内是否会发生逾期。
  - 提供可解释的特征贡献，支持风控策略制定与人工复核。
- **指标要求**：
  - 测试集 ROC-AUC ≥ 0.80 作为合格门槛，优选达到 0.90 以上。
  - 关注 Recall（召回率），确保高风险客群尽量不漏判。
- **交付要求**：清洗后的数据集、可复现代码、图表报告以及角色总结，便于课堂验收与后续迭代。

### 3. 成果评价
- XGBoost 模型在测试集达到 ROC-AUC 0.9649、F1 0.8209，超过目标值；逻辑回归基线也稳定在 ROC-AUC 0.8747。
- 特征重要性分析显示“历史逾期行为”“借款金额/历史成功金额比值”等指标作用显著，满足风控解释需要。
- 流水线支持一键重跑，并输出图表与调参记录，方便后续升级。

### 4. 后续需求展望
- 引入宏观经济指标或用户行为日志，捕捉宏观环境与实时行为变化。
- 探索分层阈值策略，对不同风险等级采取差异化定价或人工审批流程。
- 在实际业务中接入实时接口或批量评分流程，将模型嵌入贷前审批系统。
